Overview

Apa yang Akan Kamu Kerjakan?

Kamu akan mendesign 2 hal utama:

Builder Interface

Tampilan alat yang kami pakai untuk build aplikasi. Seperti Figma-nya Figma.

Component Library

Komponen-komponen yang bisa di-drag & drop. Seperti stiker/template di Canva.

Tanggung Jawab

Roadmap Kerja

Bulan 1-2

Brand Identity & Design System

  • Logo perusahaan
  • Color palette & typography
  • Icon set & guidelines
Output: Brand guideline PDF, Logo files, Figma assets
Bulan 2-4

Builder Interface Design

  • Drag & drop canvas
  • Component sidebar
  • Property panel & preview
Output: High-fidelity mockup, Interactive prototype
Bulan 3-6

Component Library

  • 20-30 komponen UI
  • Multiple variants & states
  • Responsive + dark mode
Output: Figma component library + documentation
Bulan 5-8

Template Applications

  • Template POS, CRM, Inventory
  • Template Accounting, HR
  • 5-10 halaman per template
Output: Complete Figma designs for development

Tools yang Dipakai

Learning Path

Week 1-2 Design systems, Atomic design, Responsive principles
Month 1-3 Color theory, Typography, Accessibility (WCAG)
Month 4-6 Form design, Data visualization, Micro-interactions
Month 7-12 Design tokens, Design-to-code, User testing
Success Metrics

Milestone & Target

Month 1-3
  • Brand identity selesai
  • Design system ready
  • 10 komponen designed
Month 4-6
  • Builder interface done
  • 20+ komponen completed
  • 1-2 template apps
Month 7-9
  • 3-5 template apps
  • Website design done
  • Marketing materials
Month 10-12
  • Beta customer support
  • Iterate on feedback
  • Component library v2

Typical Week

Senin Team meeting, planning, review feedback
Selasa-Kamis Design work 4-6 jam, collab dengan dev
Jumat Show & tell, documentation, planning

Flexible — Ga harus 9-5. Yang penting deadline kepegang.

FAQ

Baru pakai Figma 6 bulan?

Cukup! Yang penting paham basic. Sisanya belajar sambil jalan.

Perlu bisa coding?

Tidak wajib, tapi HTML/CSS basics akan membantu.

Belum pernah bikin design system?

Tidak masalah. Kita belajar bareng.

Requirements

Kamu Cocok Kalau...

Paham Figma dasar (components, auto-layout, variants)
Punya portfolio 3-5 project (bisa personal)
Suka design produk digital (app/web)
Detail-oriented & bisa terima feedback
Mau belajar UX, bukan cuma visual
Bisa kerja dengan developer (design handoff)
